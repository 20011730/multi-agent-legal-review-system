# 중간발표 시연 시나리오

## 시연 흐름 (공통)

```
회원가입/로그인 → /input 입력 → 로딩 → /result 토론 확인 → /verdict 최종 판정 확인 → /reviews 기록 재열람
```

## 시나리오별 상세 시연 스크립트

### 시나리오 1: 과장 광고 검토

**입력값:**
- 기업명: `AI테크 주식회사`
- 산업 분야: `IT·소프트웨어`
- 검토 유형: `마케팅·광고 문구`
- 상황 설명: `신제품 출시를 앞두고 TV 및 온라인 광고에 사용할 문구를 최종 검토합니다. 대상 고객은 일반 소비자입니다.`
- 검토 원문: `업계 최고 성능의 AI 솔루션, 경쟁사 대비 2배 빠른 처리 속도를 경험하세요`
- 참여 방식: `관찰 모드`

**기대 결과:**
- 법률 전문가: "최고", "2배" 등 과장·비교 표현 → 표시광고법 위반 가능성
- 리스크 관리자: 과징금, 시정명령, 경쟁사 대응 리스크
- 윤리 검토자: 소비자 신뢰 저하, ESG 관점
- 최종 판정: 조건부 수정 권고 / 위험 수준 HIGH

**시연 강조점:**
> "3명의 에이전트가 각각 다른 관점에서 같은 문구를 분석하고, 3라운드에 걸쳐 점점 구체적인 의견을 제시합니다."

---

### 시나리오 2: 불공정 약관 검토

**입력값:**
- 기업명: `스타트업 커머스`
- 산업 분야: `유통·물류`
- 검토 유형: `계약서·약관`
- 상황 설명: `온라인 쇼핑몰 이용약관 개정을 앞두고 법적 문제가 없는지 사전 검토합니다.`
- 검토 원문: `본 계약 체결 후 환불은 일체 불가하며, 분쟁 발생 시 당사 소재지 법원을 전속 관할로 한다`
- 참여 방식: `관찰 모드`

**시연 강조점:**
> "계약서 유형에서는 약관규제법, 전자상거래법 등 다른 법적 근거가 적용됩니다. 검토 유형에 따라 분석 관점이 달라지는 것을 보여줍니다."

---

### 시나리오 3: 긴급성 마케팅 검토

**입력값:**
- 기업명: `딜커머스`
- 산업 분야: `유통·물류`
- 검토 유형: `마케팅·광고 문구`
- 상황 설명: `연말 프로모션 광고 문구입니다. SNS 및 이메일 마케팅에 사용 예정입니다.`
- 검토 원문: `지금 가입하면 선착순 100명 한정 90% 할인! 놓치면 후회합니다`
- 참여 방식: `관찰 모드`

**시연 강조점:**
> "복수의 위험 유형(긴급성 조성 + 과장 표현)이 동시에 탐지되며, 각 에이전트가 서로 다른 위험을 지적합니다."

## 시연 시 예상 질문 & 답변

| 질문 | 답변 |
|------|------|
| AI가 실제로 판단하는 건가요? | 현재는 규칙 기반 분석 엔진이며, 향후 LLM 연동 예정입니다 |
| 법령 근거는 어디서 가져오나요? | 국가법령정보센터 Open API를 연동했으며, API 키 설정 시 실제 법령/판례가 표시됩니다 |
| 에이전트가 서로 대화하나요? | 3라운드에 걸쳐 분석 → 심화 → 권고 순으로 진행하며, 이전 라운드 의견을 참고합니다 |
| 실제 법률 자문을 대체할 수 있나요? | 참고용 분석 도구이며, 최종 판단은 법무팀/전문가 검토가 필요합니다 |
